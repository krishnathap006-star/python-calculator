# Simple Calculator Using Python

## Overview

This is my first Python mini project. It is a simple command-line calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division. I created this project to improve my understanding of Python basics and programming logic.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid input
- Prevents division by zero
- Menu-driven interface

## Technologies Used

- Python 3

## Project Structure

```
python-calculator/
├── calculator.py
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Run the following command:

```bash
python calculator.py
```
